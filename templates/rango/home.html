Introduction


This project is based on the instructions taught in the online Django Book ‘Tango with Django’ authored by ‘Leif Azzopardi’ and ‘David Maxwell’.

The project aims to form an online link directory named 'Rango' categorized under different groupings. The project uses Python 2.7 and Django 1.7 along with some other packages.

This project is actually a replica of the app ‘Rango’ taught in the above referred book. The book not only makes the concepts easy to understand but also covers minor details which some time causes novice learners to get stuck for days.

Acknowledgement


For several months I had been craving to get into the Django Web Framework and due to the unavailability of proper working tutorials I was having very hard time to actualize the concepts understood from the official documentation and other assorted tutorials.

Another problem to my learning efforts was that the tutorials available on the internet were mostly version associated and it was very difficult to be on track while using these tutorials. These tutorials were addressing different versions and were quite confusing due to the writers’ approach and their version specific focus.

Meanwhile the online book “Tango with Django” was updated to Django version 1.7 fortunately and this was the turning point for my learning curve. I read each and every character of the updated book and it took me to the fantastic journey of developing my first ever Django Site.

In a nutshell the books helped me a lot to understand the basic course of development process using Django.

I am highly grateful for both authors of this great book.
